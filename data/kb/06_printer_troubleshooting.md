- "Printer not found" or "Printer offline" error

- Print job stuck/pending indefinitely
- Printer not responding to print commands

## Solution A: Printer Offline

**Problem**: Printer shows as "Offline" in Windows settings

**Quick Fix - 5 Minutes:**
1. **Restart the printer**:
   - Locate power button on printer
   - Hold button or turn off for 10 seconds
   - Count to 10 while it shuts down
   - Turn back on and wait for startup (30-60 seconds)

2. **Restart your computer**:
   - Save all work
   - Restart Windows completely
   - Wait for full startup
   - Try printing again

3. **Check printer connection**:
   - Verify printer is connected to network (WiFi or Ethernet)
   - Look for WiFi/network icon on printer display
   - Note any error lights or messages on printer

4. **Print a test page**:
   - Windows → Settings → Devices → Printers & Scanners
   - Select your printer
   - Click "Open queue"
   - Right-click → "Printer properties"
   - Click "Print Test Page"
   - Listen for printer response

**If still offline:**
- Printer may need WiFi reconnection
- Check if WiFi password changed
- Restart WiFi router
- Contact IT support: "Escalating to IT team..."

## Solution B: Print Job Stuck

**Problem**: Print job appears in queue but never prints, blocking other jobs

**Quick Fix - 3 Minutes:**
1. **Clear the print queue**:
   - Windows → Settings → Devices → Printers & Scanners
   - Click on your printer
   - Click "Open queue"
   - Look for stuck print job
   - Right-click on stuck job → "Cancel"

2. **If queue won't clear**:
   - Press **Windows Key + R**
   - Type: `services.msc`
   - Find "Print Spooler"
   - Right-click → "Stop"
   - Wait 10 seconds
   - Right-click → "Start"

3. **Restart printer again**:
   - Power off for 10 seconds
   - Power back on
   - Try printing test page

4. **Try printing again**:
   - Open document
   - File → Print
   - Select printer
   - Click "Print"

**If job still stuck:**
- Printer may have hardware issue
- Contact IT support with printer name and model

## Solution C: Can't Find/Connect to Printer

**Problem**: Printer not appearing in available printers list

**Quick Fix - 5 Minutes:**
1. **Verify printer is on network**:
   - Check printer power
   - Verify printer connected to WiFi or Ethernet
   - Look for network indicator on printer

2. **Add printer to Windows**:
   - Windows → Settings → Devices → Printers & Scanners
   - Click "Add a printer or scanner"
   - Wait 30 seconds for scan
   - Look for your printer in the list
   - Click on it and "Add device"

3. **If printer not found in scan**:
   - Try "The printer that I want isn't listed"
   - Select "Add a printer using an IP address or hostname"
   - Enter printer IP address: **[Check printer settings or ask IT for IP]**
   - Click Next and follow wizard

4. **If still not found**:
   - Printer may need to be reset
   - Restart printer (power off 10 seconds, power on)
   - Wait 1 minute for network connection
   - Try adding again

**If still not available:**
- Printer may need IT configuration
- Contact IT support: "Escalating to IT team..."

## Solution D: Printer Not Responding

**Problem**: Printer appears connected but doesn't respond to print commands

**Quick Fix - 5 Minutes:**
1. **Verify printer status**:
   - Check printer display screen
   - Look for error lights (red, orange, blinking)
   - Note any error messages

2. **Check for common issues**:
   - **Out of paper**: Refill paper tray
   - **Paper jam**: Open all access doors and remove jammed paper
   - **Toner/ink low**: Replace cartridge if indicated
   - **Door open**: Close all covers and access doors

3. **Power cycle printer**:
   - Turn off for 10 seconds
   - Turn back on
   - Wait for startup complete

4. **Print test page** (from printer menu):
   - Use printer control panel buttons
   - Navigate to "Settings" or "Maintenance"
   - Select "Print Test Page" or "Test Print"
   - If test page prints, printer is working

**If printer still not responding:**
- Hardware may be malfunctioning
- Contact IT support with error messages

## Printer Maintenance Tips
1. **Regular cleaning**: Keep printer clean from dust
2. **Paper quality**: Use appropriate paper type/weight
3. **Toner/ink**: Keep supplies stocked to avoid empty cartridge
4. **Firmware updates**: Check for printer software updates monthly
5. **Network connection**: Ensure stable network for printer
